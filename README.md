# Report Card Generator

A simple browser-based report card generator built with HTML, CSS, and JavaScript.

## Features

- Add student information and subject scores
- Dynamically add or remove subjects
- Calculate averages and grade letters
- Generate a printable report card
- Download report cards as PDF
- Save and load student reports using local storage
- Search saved reports by student name or class
- View class statistics
- Dark mode toggle
- Export and import report data as JSON

## Files

- `index.html` — main application layout
- `app.js` — application logic
- `.gitignore` — files to ignore in Git
- `LICENSE` — project license

## Usage

1. Open `index.html` in your browser.
2. Enter student name and class.
3. Add scores for core subjects or add new subjects.
4. Click **Calculate Grades** to generate the report card.
5. Use the **Download as PDF**, **Export All Data**, or **Import Data** buttons as needed.
